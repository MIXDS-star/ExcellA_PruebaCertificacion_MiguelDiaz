# Poyecto Control de Costos en Obra – Excel + IA


**Objetivo:**
Analizar desviaciones entre presupuesto y costo real en una obra de vivienda multifamiliar.

**Herramientas:**
- Excel
- Tablas dinámicas
- Segmentadores
- Fórmulas automatizadas
- Análisis con IA

**Resultados:**
Se identificaron sobrecostos significativos en actividades estructurales y acabados, principalmente en el rubro materiales.
